# Cognitive Science and Psycholinguistics Laboratory

Website for the **Cognitive Science and Psycholinguistics Laboratory (CSPL)** at Tribhuvan University, Kirtipur, Nepal.

*Bracing the brain, minding the mind*

## About

CSPL is Nepal's first and only experimental laboratory studying how bilingualism, multilingualism, literacy, and culture shape the mind. The lab uses mousetracking and eye-tracking paradigms to investigate parallel language activation and cognitive processing.
